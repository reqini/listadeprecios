# 🔍 Guía Rápida: Buscar Productos Essen con IA

## 🎯 ¿Qué hace?

El **Buscador Global** encuentra automáticamente productos Essen en:
1. Tu catálogo interno (Google Sheets)
2. La web (MercadoLibre, Google, Bing)

Todo al mismo tiempo y en segundos.

---

## 🚀 Cómo Usar (3 Pasos)

### 1️⃣ Abrir Buscador
- Menú (☰) → "Búsqueda Global" 🔍
- O ir a: `/busqueda-global`

### 2️⃣ Escribir Búsqueda
Ejemplos que funcionan:
- `"terra"` → Encuentra todos los productos Terra
- `"sartén 28"` → Sarténes de 28cm
- `"cacerola"` → Todas las cacerolas
- `"38252866"` → Busca por código
- `"express"` → Productos de línea Express

**No necesitas escribir el nombre completo.**

### 3️⃣ Ver Resultados
Aparecen en 3 tabs:

📦 **Catálogo**: Productos de tu catálogo interno
🌐 **Web**: Productos encontrados en MercadoLibre/web
⚖️ **Comparaciones**: Precios comparados

---

1. **Usa palabras clave**: "sartén", "terra", "28cm"
2. **Códigos funcionan**: "38252866"
